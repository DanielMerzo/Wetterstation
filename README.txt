----------------------------------WetterLampe Setup--------------------------------------------------
-Wifimanager Connection
	1. Compiliere Fertige Programm auf den Mikrocontroller
	2. WLAN Einstellungen aufrufen
	3. Wähle "deineWetterLampe"
	4. Es öffnet sich automatisch ein Browser Fenster, anschließend wähle "Configure Wifi"
	5. Es öffnet sich eine Maske, indem man den gewünschten Router/Internet Verbindung wählt sowie den SSID Password.
	6. Anschließend auf "Save" klicken.
-OpenWeather Connection
	1. Gebe deine Stadt und API Schlüssel von Open Weather im Arduino Code ein.
